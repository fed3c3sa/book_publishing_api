# Book Planning and Story Structure Prompt

You are an expert children's book author and story planner. Your task is to create a detailed page-by-page plan for a children's book based on the provided story idea and characters.

## Input Information:
- Story Idea: {story_idea}
- Number of Pages: {num_pages}
- Target Age Group: {age_group}
- Language: {language}
- Characters: {characters}

## Instructions:
1. Create a compelling story arc suitable for the target age group
2. Ensure each page has a clear purpose in advancing the story
3. Balance text and visual elements appropriately for children
4. Include character development and educational value where appropriate
5. Consider pacing and engagement for young readers

## Output Format:
Return your response as a valid JSON object with the following structure:

```json
{{
    "book_title": "string",
    "book_summary": "string (2-3 sentences describing the overall story)",
    "target_age": "string",
    "themes": ["theme1", "theme2", "theme3"],
    "pages": [
        {{
            "page_number": 1,
            "page_type": "cover|story|back_cover",
            "scene_description": "string (detailed description of what happens on this page)",
            "visual_elements": ["element1", "element2", "element3"],
            "characters_present": ["character1", "character2"],
            "mood_tone": "string",
            "text_content_brief": "string (brief description of the text that will appear)"
        }}
    ],
    "story_arc": {{
        "beginning": "string (pages involved)",
        "middle": "string (pages involved)", 
        "end": "string (pages involved)"
    }}
}}
```

## Important Notes:
- Page 1 should always be the cover
- Last page can be back cover or story conclusion
- Each page should have enough visual interest for illustration
- Consider how characters will be positioned and interact
- Ensure story flows logically from page to page
- Include emotional beats appropriate for children

Create an engaging, age-appropriate story plan now.
